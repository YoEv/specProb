# 分析报告

## Task 0: 模型层面审查 - 'Pop' 流派

我们针对 'Pop' 流派训练了一个二元分类器 (Logistic Regression)，并检查了训练后模型的权重，以探究模型学到的内容。

### 核心发现

原始数据库分布正常，抽样检查Pop和Rock的spectrogram正常。

**模型未能学习到有意义的特征。**

### 权重统计数据

- **均值 (Mean):** 0.000007
- **标准差 (Std Dev):** 0.007194
- **最大值 (Max):** 0.031850
- **最小值 (Min):** -0.031502

### 结论

模型所有权重的均值、标准差、最大值和最小值都非常接近于零。这表明，对于区分 'Pop' 音乐与非 'Pop' 音乐，模型没有从频谱特征中找到任何强有力的线性信号。换句话说，简单的线性探测器不足以从这些特征中解码出流派信息。
